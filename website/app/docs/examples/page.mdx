# Examples

All snippets assume the functions are imported from the package root:

```js
import { diff, patch } from "@opentf/obj-diff";
```

## 1. Basic Objects
```js
const a = { a: 1, b: 2 };
const b = { a: 2, c: 3 };

diff(a, b);
/*
[
  { type: 2, path: ["a"], value: 2 },
  { type: 0, path: ["b"] },
  { type: 1, path: ["c"], value: 3 }
]
*/
```

## 2. Nested Structures
```js
const a = { foo: { bar: [1, 2] } };
const b = { foo: { bar: [1] } };

const d = diff(a, b);
const res = patch(a, d); // res is deep equal to b
```

## 3. Compact Array Diffs
Array edits produce shortest edit scripts — one op per actual insert/remove.

```js
diff([1, 2, 3], [0, 1, 2, 3]);
//=> [{ type: 3, path: [0], value: 0 }]   // INSERTED at index 0

diff([1, 2, 3, 4], [1, 4]);
//=> [{ type: 4, path: [1] }, { type: 4, path: [1] }]   // REMOVED (application-time indexes)
```

## 4. ES6 Map & Set Support
Natively diff and patch modern collections.

```js
const a = new Set([1, 2]);
const b = new Set([2, 3]);

diff(a, b);
/*
[
  { type: 2, path: [0], value: 2 },
  { type: 2, path: [1], value: 3 }
]
*/
```

## 5. Circular Reference Safety
Safe comparison of recursive objects without infinite loops.

```js
const a = { id: 1 };
a.self = a;
const b = { id: 2 };
b.self = b;

diff(a, b); 
// Output: [{ type: 2, path: ["id"], value: 2 }]
```

## 6. Native TypedArray Performance
Accurately diffs large chunks of binary data incredibly fast.

```js
const a = new Uint8Array([1, 2, 3]);
const b = new Uint8Array([1, 4, 3]);

diff(a, b); 
// Output: [{ type: 2, path: [1], value: 4 }]
```
