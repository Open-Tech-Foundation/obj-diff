# Getting Started

This page is rendered by `app/docs/page.mdx` and wrapped by
`app/docs/layout.jsx`, which uses `DocsLayout` from `@opentf/web-docs`.

## Navigation

The top navbar links here from the landing page. The sidebar is generated from the
docs route tree, and `app/docs/_meta.js` controls the page labels and order.

## Table Of Contents

Headings on this page are collected into the right-side table of contents. Add more
`##` and `###` headings to see it grow.

## Edit Content

Change this MDX file and run the dev server:

```bash
bun run dev
```
